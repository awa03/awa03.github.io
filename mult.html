Multiple Inheritance

Multiple Inheritance is not supported by all programming languages (java for example). Multiple Inheritance simply allows us to inherit properties from multiple parent classes. For instance a car could be a part of the sales parent class and vehicles parent class.

Example

The course example given is of older systems used to use multiple inheritance for implementation of stream libraries for file I/O. This however has been replaced for the template class method

ifstream--input file stream class
ofstream--output file stream class

In the older implementations, there was a class called fstreambase, which contained member functions relating to file-handling operations, such as open() and close() (for opening and closing files).

Code Example

class ifstream : public fstreambase, public istream
{ ..... };
Similarly, ofstream was inherited from ostream (output features) and fstreambase (file features)
class ofstream : public fstreambase, public ostream
{ ..... };

Multiple Inheritance is not an necessary feature of object-oriented programming
